# Sources (raw uploads)

High-resolution originals that the game-ready sprites were processed from. **Not loaded
by the game** — kept only so a sprite can be re-processed (re-trimmed, re-scaled, matte
re-cleaned) without regenerating the art.

Organized to mirror the game folders, e.g. `sources/buildings/<name>.png`. Filenames here
are the raw upload names (`*_final`, `*_v2`, …); the processed output uses the game's
internal `type` name in the sibling `assets/<kind>/` folder.
