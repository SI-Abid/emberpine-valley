# Animal sprites

Pen/pasture animals, replacing the 16–32px atlas frames. Transparent PNG, trimmed,
base flush to the bottom, no baked shadow.

## Filenames
- `sheep.png`
- `pig.png`
- `cow.png`
- `chicken.png`

## Facing
A single right-facing sprite is fine — the game flips it horizontally for the other
direction, same as it does for characters. If you make a walk animation later we can
wire a small strip.
